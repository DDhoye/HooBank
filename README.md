HOOBANK

ABOUT THE PROJECT:
Hoobank is a modern bank web application , built with react and tailwind css. 
The website is designed to provide users with accurate and upto date information about 
the key metrics of the bank.

BUILT WITH:
React Js
Tailwind CSS
